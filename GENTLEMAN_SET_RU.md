# Джентльменский набор для Meizu Note 21 с Root-правами (Magisk)

После получения Root-прав и разблокировки загрузчика на Android 14 (Flyme OS 14) открывается доступ к глубокой настройке системы. Ниже представлен базовый набор модулей, приложений и твиков, необходимых для комфортного использования устройства, скрытия Root от банковских приложений и блокировки рекламы.

---

## 1. Системные модули Magisk (прошиваются через вкладку «Модули» в Magisk)

### [Play Integrity Fix](https://github.com/chiteroman/PlayIntegrityFix)
*   **Зачем нужен**: Google блокирует работу бесконтактной оплаты (Google Pay / Wallet), работу банковских приложений (Сбербанк, Альфа-Банк и др.) и некоторых игр (например, Pokémon GO) на устройствах с разблокированным загрузчиком. Этот модуль подменяет отпечатки устройства на лету и возвращает проверку сертификации Google Play Integrity в состояние «Пройдено».
*   **Как использовать**: Скачайте `.zip` файл релиза, установите его в Magisk -> Модули -> Установить из хранилища, затем перезагрузите устройство.

### [Zygisk Next](https://github.com/Dr-TSNG/ZygiskNext) (если требуется альтернатива встроенному Zygisk)
*   **Зачем нужен**: Высокопроизводительная реализация Zygisk, работающая в обход стандартных ограничений Magisk. Улучшает скрытие Root-прав и стабильность работы модулей.
*   **Ссылка**: [Zygisk Next на GitHub](https://github.com/Dr-TSNG/ZygiskNext)

---

## 2. Модули Xposed / LSPosed (требуют активации Zygisk в Magisk)

Для работы Xposed-модулей сначала установите актуальный релиз менеджера **LSPosed** для Zygisk:
*   [LSPosed (JingMatrix Fork) на GitHub](https://github.com/JingMatrix/LSPosed) (активная ветка с поддержкой Android 14).

### [Hide My Applist](https://github.com/Dr-TSNG/Hide-My-Applist) (Модуль LSPosed)
*   **Зачем нужен**: Многие приложения (банковские клиенты, игры с защитой от читов) ищут на телефоне установленные приложения Magisk, Termux, LSPosed и блокируют запуск, даже если сам Root от них скрыт. Этот модуль позволяет создать «белый список» и полностью скрыть факт установки Magisk от выбранных приложений.

### [CorePatch](https://github.com/LSPosed/CorePatch) (Модуль LSPosed)
*   **Зачем нужен**: Отключает системную проверку подписи APK-файлов. Позволяет устанавливать модифицированные приложения поверх оригинальных без потери данных, понижать версию приложений и отключать другие ограничения установщика пакетов Android.

## 3. Полезные приложения с Root-доступом

### [AdAway](https://adaway.org/)
*   **Зачем нужно**: Лучший блокировщик рекламы для Android. Работает на уровне системы, перезаписывая файл `/etc/hosts`. Блокирует рекламу абсолютно во всех браузерах, приложениях и играх. Не расходует батарею, так как не висит в памяти в качестве локального VPN.
*   **Ссылка**: [Скачать AdAway на F-Droid](https://f-droid.org/packages/org.adaway/)

### [MiXplorer](https://forum.xda-developers.com/t/app-2-2-mixplorer-v6-x-released-fully-featured-file-manager.1523691/)
*   **Зачем нужно**: Один из лучших и многофункциональных файловых менеджеров для Android с полной поддержкой Root-доступа. Позволяет редактировать системные конфигурации (например, `/system/build.prop`), менять права файлов (chmod/chown) и просматривать любые закрытые разделы памяти.
*   **Ссылка**: [Официальная ветка на XDA](https://forum.xda-developers.com/t/app-2-2-mixplorer-v6-x-released-fully-featured-file-manager.1523691/)

### [Termux](https://termux.dev/)
*   **Зачем нужно**: Полноценный эмулятор терминала со встроенной средой Linux. Позволяет выполнять bash-скрипты, системные команды Linux и администрировать систему прямо с телефона с помощью команды `su`.
*   **Ссылка**: [Скачать Termux на F-Droid](https://f-droid.org/packages/termux/)

### [Shizuku](https://shizuku.rikka.app/) + [App Ops](https://play.google.com/store/apps/details?id=web.carisok.appops)
*   **Зачем нужно**: Shizuku использует права root для предоставления системного API другим приложениям. В связке с App Ops это позволяет детально настраивать скрытые разрешения для любых приложений (например, запретить приложению читать буфер обмена или получать доступ к буферу без уведомления системы).
